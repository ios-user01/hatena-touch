What is Hatena touch?

This is an application for much more easier to use web service of Hatena Inc. in Japan.

NOTE: It is necessary an account of Hatena to use all functions of this application.

The Hatena touch application includes the ability to:

Hatena Diary (Blog)

  • Publish new post.
  • Publish draft.
  • Read, Edit, Delete backnumbers.
  • Read, Edit, Delete drafts.
  • Help of Hatena syntax.

Hatena Bookmark
(Social Bookmark)

  • Read "Hot Entries" (RSS Reader).
  • Read comment of the site.
  • Read, Edit, Delete your bookmarks.
